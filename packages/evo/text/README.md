# Text component for Angular

## Installation

```
$ ng add @ngx-kit/ui-text
```


## Usage

```html
<input type="text" uiText>
<textarea uiText [autoResize]="true"></textarea>
```


## Options

### `[uiText]`

none

### `textarea[uiText]`

* `[autoResize]`
